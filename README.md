『扯线木偶』界面自动化应用编程接口(UIAutomation API)
==

以UIA技术实现界面操作可编程接口，突破量化交易桎梏。
--
**puppet扯线木偶目前仅适用于独立交易端核新版(即THS)，需配合交易框架(hikyuu, quantaxis, rqalpha, vnpy, etc)使用。**

免配置安装，运行前需要先自行登录交易客户端。
